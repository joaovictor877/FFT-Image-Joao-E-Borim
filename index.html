Codificador de Imagem com FFT 2D

Programa simples para codificar e reconstruir imagem usando frequência

Como funciona (resumo)

  1. Carrega a imagem e redimensiona para 128x128.
  2. Aplica FFT 2D para ir ao domínio da frequência.
  3. Aplica uma cifra fixa e reversível nos coeficientes.
  4. Aplica IFFT para reconstruir a imagem.
